# Seamra Works

Static source for `https://seamraworks.com`.

Seamra Works publishes focused developer tools for understanding the shape of polyglot software systems. Our first product is AtlasArc, an IntelliJ IDEA plugin for exploring Java, Kotlin, and TypeScript architecture through connected views of dependencies, metrics, heatmaps, subsystem boundaries, and cycle health. For product details, documentation, and downloads, visit [AtlasArc.io](https://atlasarc.io).

## Local Preview

Open `index.html` directly in a browser. There is no build step.
